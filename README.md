# BiteTrack 🥗

Tracker personal de nutriție — calorii, macros, apă, istoric 7 zile.

## Stack
- React 18 + Vite
- Recharts pentru grafice
- localStorage pentru persistență

## Instalare locală
```bash
npm install
npm run dev
```

## Deploy pe Vercel
1. Push repo pe GitHub
2. Conectează repo pe vercel.com
3. Vercel detectează automat Vite — click Deploy
